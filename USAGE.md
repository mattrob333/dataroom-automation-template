- Claude Code installed and running

## Basic Usage

### Simple Command

Open Claude Code in this directory and say:

```
Generate a data room for [Company Name]
```

**Examples**:
```
Generate a data room for Stripe
Generate a data room for https://www.notion.so
Create a comprehensive data room for Acme Corp (acmecorp.com)
```

## Tips for Best Results

### 1. Provide Context
The more information you provide, the better:
- Company stage (seed, Series A, etc.)
- Industry/sector
- Known funding information
- Specific areas of interest

### 2. Iterate
- Review generated documents
- Ask for clarifications
- Request deeper analysis
- Refine iteratively

### 3. Verify Information
- Check key facts
- Validate financial figures
- Confirm dates and events
- Cross-reference sources

## Common Requests

### Make it shorter
```
Regenerate the executive summary but make it 1 page instead of 3
```

### Add more detail
```
The financial section needs more detail - add revenue breakdown
```

### Update with new info
```
Company just raised a Series B - update the data room
```

## Troubleshooting

**Issue**: Not enough information found  
**Solution**: Provide additional sources or context

**Issue**: Generated content is too generic  
**Solution**: Provide more specific context about the company

**Issue**: Inaccurate information  
**Solution**: Correct Claude and provide accurate sources
