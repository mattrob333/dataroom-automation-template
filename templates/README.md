# Document Templates

This directory contains markdown templates for all data room documents.

## Usage

Templates use `{{VARIABLE}}` syntax for placeholders that should be replaced with actual data.

## Available Templates

### Executive Documents
- `executive_summary.md` - 2-page executive summary
- `company_overview.md` - Detailed company overview

### Business Documents
- `business_model.md` - Business model canvas and description
- `market_analysis.md` - Market size, trends, and analysis
- `competitive_analysis.md` - Competitive landscape

### Financial Documents
- `financial_model.md` - Financial projections structure
- `unit_economics.md` - Unit economics breakdown

### Legal Documents
- `corporate_structure.md` - Legal entity structure
- `risk_factors.md` - Risk assessment

### Technical Documents
- `technical_architecture.md` - System architecture
- `product_roadmap.md` - Product development roadmap

## Customization

1. Copy any template to your working directory
2. Replace `{{VARIABLES}}` with actual data
3. Customize sections as needed
4. Convert to PDF for final data room
