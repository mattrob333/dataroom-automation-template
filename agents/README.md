# Sub-Agents Configuration

This directory contains specialized agent configurations for the data room generation workflow.

## Agent Types

1. **Research Agent** - Company information gathering
2. **Financial Agent** - Financial data analysis
3. **Legal Agent** - Legal and compliance documentation
4. **Technical Agent** - Technical assessment

Each agent file contains:
- Agent purpose and scope
- Required tools and capabilities
- Prompt templates
- Output specifications
- Quality criteria
